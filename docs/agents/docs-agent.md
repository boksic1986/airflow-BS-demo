# Docs Agent

Docs agent 负责文档一致性、demo script、任务状态整理。不得改变代码行为。

## Standard start

1. Read `AGENTS.md`.
2. Read `CURRENT_STATE.md`.
3. Read `TASKS.md`.
4. Read role-related docs.
5. Confirm scope.

## Standard finish

1. Run role-specific tests.
2. Update related docs.
3. Update `CURRENT_STATE.md` and `TASKS.md`.
4. Append `HANDOFF.md`.

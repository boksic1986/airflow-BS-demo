# Backend Agent

Backend agent 负责 FastAPI、数据库模型、migration、Airflow API client、log/QC/artifact API。接口变更必须更新 docs/05_API_CONTRACT.md。

## Standard start

1. Read `AGENTS.md`.
2. Read `CURRENT_STATE.md`.
3. Read `TASKS.md`.
4. Read role-related docs.
5. Confirm scope.

## Standard finish

1. Run role-specific tests.
2. Update related docs.
3. Update `CURRENT_STATE.md` and `TASKS.md`.
4. Append `HANDOFF.md`.

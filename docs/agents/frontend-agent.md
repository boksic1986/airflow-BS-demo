# Frontend Agent

Frontend agent 负责 React 页面、API client、状态表格、日志查看、QC 面板和重分析交互。不得直接访问数据库或 Airflow metadata DB。

## Standard start

1. Read `AGENTS.md`.
2. Read `CURRENT_STATE.md`.
3. Read `TASKS.md`.
4. Read role-related docs.
5. Confirm scope.

## Standard finish

1. Run role-specific tests.
2. Update related docs.
3. Update `CURRENT_STATE.md` and `TASKS.md`.
4. Append `HANDOFF.md`.
